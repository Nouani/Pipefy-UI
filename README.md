## How To Use

#### Installation

In your gitbash terminal:
```
>   git clone https://github.com/Nouani/Pipefy-UI.git
```

On your open machine terminal in the Pipefy-UI folder:
```
>   cd Pipefy-UI
```
```
>   yarn 
    or 
>   npm
```

#### Start up

```
>   yarn start 
    or 
>   npm run start
```

#### Note: You must have react.js installed globally.
